Подтягивание на перекладине –

общепринятый показатель физической подготовленности.

Каждый день мы "штурмовали" турник, да еще по

нескольку раз.

Очень здорово, что почти все, кто уже умел подтягиваться,

улучшил свои результаты.

Нельзя не отметить безусловных лидеров

– Толика и Дениса.

Более, чем в два раза также улучшили

свои показатели Саша и Антон.

Игорь и Максим – просто молодцы, старайтесь и дальше!

 

Володя

Инструктор лагеря по стрелковой подготовке (устройство АК). Победитель в соревнованиях по стрельбе из винтовки, ориентированию, горному биатлону, призовые места по летнему и ориент-биатлону, стрельбе из винтовки, спутниковой навигации.

Влад

Инструктор лагеря по горной подготовке (узлы).

Победитель в соревнованиях по стрельбе из AEG в двух упражнениях, спутниковой навигации, призовое место по ориентированию. Выполнил норматив Минобороны по сборке-разборке АК на "отлично".

 

Саша

Победитель в соревнованиях по ориентированию и летнему биатлону, призовые места по стрельбе из пистолета и AEG, горному биатлону, GPS-навигации,  скоростной вязке узлов,  выполнил норматив Минобороны по разборке АК.

 

                                        Денис

 

Победитель в соревнованиях по стрельбе из винтовки, определении расстояний, призовые места в соревнованиях по стрельбе из винтовки, пистолета, скоростной вязке узлов, второе место по подтягиванию на перекладине – 15 раз!

Дима

Победитель в соревнованиях по скоростной стрельбе из AEG, GPS-навигации, ориент-биатлону, призовые места по стрельбе из  винтовки и  AEG в двух упражнениях, скоростной вязке узлов, ориентированию, летнему биатлону, 

Толик

Победитель в соревновании по горному биатлону, призовые места в стрельбе из пистолета, скоростной стрельбе из AEG, ориентированию, ориент-биатлону, определении расстояний, первое место по подтягиванию – 18 раз!

 

Дима
