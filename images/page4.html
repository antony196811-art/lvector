Наши результаты 

Лето 2008 года 

 

 

 

 

Максим

 

Победитель в соревнованиях по стрельбе из винтовки стоя, AEG с колена, призовые места в стрельбе из винтовки лежа, выполнил норматив Минобороны по сборке-разборке АК и  ПМ на "отлично", зачет по горной подготовке (узлы) на "отлично". В соревнованиях по подтягиванию занял 1 место (13 раз).

Егор

 

Победитель в соревновании по стрельбе из винтовки (лежа) и определению расстояний, выполнил нормативы Минобороны по сборке-разборке АК и ПМ, сдал зачет по медицинской подготовке на отлично, призовые места по скоростной стрельбе из AEG, ориентирование (желт. трек), GPS-навигация (база).

 

 

      

Никита

 

Победитель в соревнованиях по горному биатлону, призовые места в соревнованиях  по скоростной стрельбе из AEG, применению ПМ, летнему и ориентбиатлону, выполнил нормативы Минобороны по сборке-разборке АК и ПМ на "отлично"

Инструкторы

Володя

Инструктор лагеря по стрелковой подготовке (устройство АК и ПМ)

 

 

 

Настя

 

Победитель в соревнованиях по стрельбе из пистолета, скоростной стрельбе из AEG, определению расстояния. Призовые места в стрельбе из винтовки стоя, AEG с колена, применение ПМ, летнем биатлоне, ориентировании (желтый трек). 

Кирилл

 

Победитель в соревнованиях по стрельбе из пистолета и GPS-поиск (лес), призовые места в соревнованиях по стрельбе  из винтовки стоя и  AEG колена,  выполнил норматив Минобороны по сборке-разборке АК и  ПМ.

 

 

 

Сергей

 

Призовые места в соревнованиях по стрельбе из винтовки лежа, AEG с колена, GPS-поиск (лес), выполнил норматив Минобороны по сборке-разборке ПМ.

 

 

Влад

Инструктор лагеря по горной подготовке (узлы)

 

Дима

Получил в этом году инструкторский галстук.

Инструктор лагеря по горной подготовке (промальп)

 

 

Таня

 

Победитель в соревнованиях по летнему, ориент- и горному биатлону, ориентированию (красный трек), стрельба из винтовки стоя, определение расстояний, сдала на "отлично" зачет по медицинской и горной (узлы) подготовке, призовые места в стрельбе из винтовки лежа, AEG с колена, применение ПМ.

 

 

 

Женя

Победитель в соревнованиях по стрельбе из винтовки стоя, выполнил норматив Минобороны  по сборке-разборке АК на "хорошо" и ПМ на "отлично", сдал зачет по медподготовке на "отлично", призовые места в стрельбе из винтовки лежа, пистолета, GPS-поиск (база),  применение ПМ.
